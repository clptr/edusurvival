# 🌿 EduSurvival – Website Edukasi Survival

EduSurvival adalah website edukasi interaktif yang menyediakan pembelajaran dasar mengenai keterampilan bertahan hidup (survival skills) dalam berbagai situasi darurat.

---

## 🎯 Tujuan Project

- Meningkatkan kesadaran tentang pentingnya survival skill  
- Memberikan panduan praktis menghadapi situasi darurat  
- Menyediakan media pembelajaran yang mudah dipahami  

---

## ✨ Features
- Home
- About Us
- Survival Skill (first aid, disaster survival, outdoor survival)

---

## 🧑‍💻 Tech Stack

- HTML  
- CSS  
- JavaScript  

---

## ⚙️ Setup

1. Download / clone repository  
2. Buka folder project  
3. Jalankan file `index.html` di browser  

---

## 🏗️ Architecture

User → Browser → Website → Konten ditampilkan

Website ini menggunakan arsitektur frontend sederhana tanpa backend.
